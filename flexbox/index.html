min and max content

  1. Comment "display:flex;"
  2. Add "width:max-content" to each list item, each item occupies a new line and the width is the length of its content
  3. Change max to min, some items occupy multiple lines so that the width is reduced to its min

By default, flex uses max content.

default behavior of flex

  1. Delete the min content and uncomment "display:flex;"
  2. Now if we shrink the webpage, the third item will wrap first to adapt to smaller container
  3. When all items reach their min length (min content), and we continue shrink the window, the items will overflow

flex-shrink

This property is 1 by default, means it's turned on, so items can shrink when there isn't enough room

gap

Add "gap: 1em;" to container, each item will have a gap of 1em with adjacent item.

flex-grow

Add "flex-grow: 1;" to item, this code will allocate extra space to each item.

flex-wrap

Add "flex-wrap:wrap;" to the container, so when the container is too small, content will wrap so we don't have overflow anymore.

We can comment flex-grow and flex-shrink and only use flex-wrap to avoid overflow.

justify-content command

Comment flex-wrap.

Justify-content only works when there is extra space.

Add "justify-content:center;" to container. Put all items centered. We have flex-start, flex-end, space-between, space-around, space-evenly.If we've already set items to have a flex-grow as 1, then justify-content doesn't work because all extra space are allocated to each item.

We can also change main axis direction to column. Then to see justify-content works, we need to set height for the container. The author suggests set a min-height instead of actual height.
