Initial status

Add "display:flex;" and "gap: 2rem;" to columns class, we can separate three items but they are of different width.

That's because by default, flex-basis is auto, flex will calculate best width for each item. The third item has max length so its auto width is largest.

To solve this problem, we need to set flex: 1. That means we set the three parameter to 1 1 0. So, each item is initialized with a width of 0 and expand to same width.

flex-direction

Default flex-direction is horizontal, or row. This attribute can be set in containers.

Change flex-direction to column. The items are arranged vertically. Question, now the third item is higher than others even when we set flex to 1 1 0. Explanation: If we set the height of columns to a larger value, the three items' height is the same. If we don't specify the height, then the flex-basis is auto by default and all three items' height is increasing in same scale but the third item has more content so even previous two stop growing, it still need to increase height until all content is included.

When we add a @media code block in css, the flex direction changes according to the web page size. If it's less than the threshold, flex direction is column, otherwise it's row.

Alignment

The default of align-item is stretch. Options are flex-start, flex-end, center, stretch.

Align item can be used on single item. Use align-self.
